# CSV-JSON Converter
CSV-JSON Converter - приложение для преобразования данных из формата CSV в JSON с ориентацией на колоночную архитектуру.
Основная цель - создать конвертер данных для работы с ClickHouse.

# Возможности
1. Загрузка CSV-файлов.
2. Отображение содержимого файла (предпросмотр).
3. Преобразование в JSON с предпросмотром результата.
4. Возможность сохранения результата.

# Стек
C#, .NET, WPF, CSV, CSVHelper, JSON, Newtonsoft.Json.
